DiscoverEd Search Point

This DiscoverEd Search Point can be used to search the Library's physical collections.


Use this DiscoverEd Search Point to find:


  • Books
  • Journal Titles
  • Audio-visual resources
  • Other physical materials held in the Library

There is no option at this DiscoverEd Search Point to:


  • Sign in to your Library Account
  • Search for (or access) online resources such as e-books and e-journal and database content

To access your Library Account or search for online resources sign into DiscoverEd using one of the Library's open access PCs or MACs or using your own device.
